La composition de Ebook

Je vais vous montrer de quoi sera composé le ebook.
Comment commencer
Dans la première partie de ce livre, je vais vous expliquer comment comment commencer le street workout selon votre morphologie, les technique d'entrainement et la nutrition
Apprendre les figures
Je vais vous montrer les techniques que j'ai utilisées et des informations sur les différents type de figure pour pouvoir les apprendre dans les meilleurs conditions possible.
Mon avis
Dans cette catégorie, je donnerai mon avis sur les "figures de bases" qui sont des prérequis pour apprendre le street workout, je donnerai aussi ma liste sur quelles figures apprendre au debut.
Le handstand
Le handstand la figure qui défini les bases du streetworkout, je donnerais quelques informations pour pouvoir apprendre le handstand mais je vous renverrais vers un programme complet pour apprendre le handstand, si vous le souhaitez vraiment.
Un lexique
Pour être sur de la comprehension de mon message par tout le monde, vous pourrez trouver un lexique de pleins de figure de base ainsi que un lexique général du vocabulaire du street workout.

De vrais avis d'utilisateurs !
